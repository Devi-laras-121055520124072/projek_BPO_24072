Sistem Data Kendaraan



> Deskripsi

Sistem Data Kendaraan adalah program berbasis Python yang dibuat untuk memenuhi tugas

Mata Kuliah Pemrograman Berorientasi Objek (PBO). Program ini digunakan untuk mengelola

data kendaraan seperti mobil dan motor dengan menerapkan konsep Object Oriented Programming (OOP).



> Tujuan

\- Memahami konsep Class dan Object

\- Menerapkan Inheritance dalam program Python

\- Membuat program berbasis menu

\- Mengelola data menggunakan struktur list



> Konsep PBO yang Digunakan

\- Class dan Object

\- Inheritance

\- Function

\- Percabangan (if-elif-else)

\- Perulanga



> Fitur Program

\- Menambah data mobil

\- Menambah data motor

\- Menampilkan seluruh data kendaraan

\- Keluar dari program
